# Guessing Game

This project is part of Task 02 for the Software Development Internship at Prodigy InfoTech.

## Description
The program generates a random number and allows the user to guess it.
It provides feedback for each guess and displays the number of attempts once the correct number is guessed.

## Technologies Used
- Python

## How to Run
1. Run the Python file
2. Enter your guess
3. Follow hints (Too high / Too low)
4. Guess correctly to win
